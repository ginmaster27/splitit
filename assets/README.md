# Assets

Use this folder for app visual assets.

- `icons/`: app icons, adaptive icons, notification icons, and other platform icon files
- `favicon/`: web favicon files such as `favicon.png`, `favicon.ico`, and browser manifest icons

When the final files are uploaded, wire them into `app.json` using Expo's `icon`, `android.adaptiveIcon`, `ios.icon`, and `web.favicon` fields as needed.
